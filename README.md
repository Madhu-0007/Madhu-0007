## 🚀 Featured Projects

### 🎥 ERA-MIL — Can an AI spot a crime before you do?
A multi-instance learning system that watches surveillance footage and flags anomalies — no frame-level labels needed, just weak video-level supervision.

<br>

### 🤟 Breaking the STEM-to-Sign Language Barrier
Deaf students shouldn't have to choose between understanding a physics lecture and following along in real time. This system translates spoken/written STEM content — including diagrams and OCR'd equations — into live sign language avatars.

<br>

### ❤️ Predicting Heart Disease with Quantum Machine Learning
What happens when you swap a classical SVM kernel for a quantum one? This is the code behind that question — reimagining a classic prediction problem through a quantum lens.
